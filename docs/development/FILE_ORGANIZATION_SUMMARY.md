## Best Practices

### File Placement Rules

1. **Test files** → `tests/{component}/test_*.cpp` or `tests/integration/test_*.sh`
2. **Scripts** → `scripts/*.sh`
3. **Documentation** → `docs/{category}/*.md` (except `README.md` in root)
4. **Build config** → `CMakeLists.txt` stays in root and subdirectories
5. **Build artifacts** → `build/` directory only (never in root)

### CMake Usage

```bash
# ✅ CORRECT:
mkdir -p build && cd build
cmake ..
make

# ✅ CORRECT (using script):
./scripts/build.sh

# ❌ WRONG (generates files in root):
cmake .
```
